ตัวจำลองข้อมูลเซนเซอร์ (Built-in IoT Simulator)

ใช้สำหรับทดสอบการทำงานเรียลไทม์โดยไม่ต้องต่ออุปกรณ์ฮาร์ดแวร์จริง

ปรับระดับความชื้นจำลอง: 45.0%

การส่งข้อมูลจาก ESP32 / Arduino (IoT Backend API)

อุปกรณ์ IoT สามารถส่งค่าความชื้นผ่าน HTTP POST JSON มาที่ Backend ได้ทันที:

POST http://localhost:3000/api/moisture Content-Type: application/json { "sensor_id": "zone-1", "moisture": 52.4, "temperature": 29.5, "humidity": 60.0 }
